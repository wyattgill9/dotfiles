# Dotfiles ✨
A collection of my personal configurations for both **NixOS** and **Windows** systems.  
Feel free to explore, take inspiration, or adapt parts of these files to your own setup!  

> **Note**: These dotfiles aren't meant to be cloned and used as-is. Instead, they serve as a resource to reference or pick specific configurations. If you find them helpful, a shoutout or credit would be appreciated! 😊  

---

### Getting Started  
1. Browse through the configurations in their respective directories:  
   - **`nixos/`** for NixOS configs  
   - **`windows/`** for Windows configs  
2. Copy the files or snippets you need.  
3. Adjust them to suit your environment.  

Happy tinkering! 🚀
